# كيفية تشغيل النظام

## الطريقة 1: استخدام السكريبت (موصى بها)

### على Windows:
1. انقر بزر الماوس الأيمن على `start-simple.bat`
2. اختر "Run as administrator" (تشغيل كمسؤول)
3. انتظر حتى تفتح نوافذ CMD للخوادم
4. سيتم فتح المتصفح تلقائياً

### أو باستخدام PowerShell:
```powershell
.\check-and-start.ps1
```

## الطريقة 2: التشغيل اليدوي

### 1. افتح Terminal (PowerShell أو CMD)

### 2. تأكد من وجود Node.js:
```bash
node --version
npm --version
```

إذا لم يكن موجوداً، قم بتثبيته من: https://nodejs.org

### 3. ثبّت الحزم:
```bash
npm install --legacy-peer-deps
```

### 4. شغّل Backend (في نافذة Terminal منفصلة):
```bash
npm run server
```

### 5. شغّل Frontend (في نافذة Terminal أخرى):
```bash
npm run dev
```

### 6. افتح المتصفح:
انتقل إلى: http://localhost:5173
