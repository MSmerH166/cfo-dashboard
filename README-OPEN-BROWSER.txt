============================================
كيفية فتح النظام في المتصفح
============================================

الطريقة 1: استخدام السكريبت
----------------------------
انقر نقراً مزدوجاً على ملف: open-browser.bat

الطريقة 2: فتح يدوي
-------------------
1. افتح أي متصفح (Chrome, Edge, Firefox)
2. اكتب في شريط العنوان:
   http://localhost:5173
3. اضغط Enter

الطريقة 3: من Terminal
----------------------
اكتب الأمر التالي:
start http://localhost:5173

أو:
start chrome http://localhost:5173
start msedge http://localhost:5173
